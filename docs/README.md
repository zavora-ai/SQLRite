# SQLRite Documentation

This directory contains the product and operator documentation for SQLRite.

## Start Here

| Need | Document |
|---|---|
| Product overview and first-run workflow | `README.md` |
| SQL retrieval syntax and examples | `docs/sql_cookbook.md` |
| Migration from SQLite, pgvector, libSQL, and API-first vector exports | `docs/migrations/` |
| Operations and deployment runbooks | `docs/runbooks/` |
| Security model and compliance posture | `docs/security/` |
| HA and replication architecture | `docs/architecture/` |
| Release notes and policy | `docs/releases/`, `docs/release_policy.md` |

## Public vs Internal Material

- `docs/` is for stable product documentation.
- `examples/` contains runnable usage examples.
- `benchmarks/` contains benchmark profiles and curated benchmark material.
- `project_plan/` contains internal planning, sprint execution records, and release evidence.

If you are looking for historical roadmap or benchmark-status working material, see:

- `project_plan/strategy/`
- `benchmarks/status.md`
